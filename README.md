# HTML & CSS Learning Journey

This repository contains my beginner-level HTML and CSS practice completed as part of my QA learning journey.

The goal of this repository is to build basic understanding of web page structure, HTML elements, forms, tables, media elements, and simple CSS concepts. These skills support my development as a future QA tester by helping me better understand how web applications are structured and tested.

## What I Practiced

* basic HTML document structure
* headings, paragraphs, links and images
* forms and input fields
* tables
* audio and video elements
* semantic HTML basics
* basic content structure
* beginner web page exercises

## Repository Content

This repository includes practice exercises and small beginner projects from learning platforms such as Codecademy and Codefinity.

The exercises are organized into separate folders by platform and topic.
