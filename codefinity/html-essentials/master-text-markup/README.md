# Master Text Markup

Beginner HTML challenge from Codefinity.

## Overview

In this challenge, I created a simple profile page using basic HTML text elements.

## What I Practiced

* Basic HTML document structure
* Headings: `h1`, `h2`, `h3`
* Paragraphs with `p`
* Page title and charset metadata

## Project Content

The page includes:

* Name as the main heading
* Bio section
* Hobbies section
* Skills section

## What I Learned

I learned how to organize text on a webpage using headings and paragraphs.
