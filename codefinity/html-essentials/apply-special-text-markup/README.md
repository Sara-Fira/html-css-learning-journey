# Apply Special Text Markup

Beginner HTML challenge from Codefinity.

## Overview

In this challenge, I practiced using special HTML text markup and simple interactive elements.

## What I Practiced

* Bold/important text with `strong`
* Emphasized text with `em`
* Underlined text with `u`
* Strikethrough text with `s`
* Links with `a`
* Phone and email links
* A basic button with `button`

## What I Learned

I learned how to format text in HTML and create different types of links, including website, phone, and email links.
