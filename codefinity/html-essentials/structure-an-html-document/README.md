# Structure an HTML Document

This is a beginner HTML challenge from Codefinity.

## Project Overview

The goal of this challenge was to practice creating the basic structure of an HTML document.

The task was to fill in the missing parts of an HTML code block and make sure the document had the correct basic structure.

## What I Practiced

* Writing the `<!DOCTYPE html>` declaration
* Creating the `<html>` element
* Adding the `<head>` section
* Adding the `<body>` section
* Understanding the basic layout of an HTML document

## Basic HTML Structure

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
```

## What I Learned

I learned that every HTML document should have a clear structure. The `head` section contains information about the page, while the `body` section contains the visible content displayed in the browser.
