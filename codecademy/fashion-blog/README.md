# Fashion Blog

A beginner HTML project created as part of the Codecademy **Learn HTML: Fundamentals** course.

## Project description

This project is a simple fashion blog page called **Everyday with Isa**.
The page was built using basic HTML structure and elements.

## What I practiced

* HTML document structure
* Headings and paragraphs
* Images
* Links
* Unordered lists
* `div` sections
* `id` attributes
* Internal page links
* Strong text formatting

## Features

* Blog post about New York Fashion Week
* Profile image linked to the contact section
* Related content list
* External link opening in a new tab
* Contact section with email, phone, and address

## Status

Completed as part of my HTML learning journey.
